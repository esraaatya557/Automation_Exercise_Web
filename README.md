# Automation Exercise Web
https://automationexercise.com

## Description
- Lightweight web UI automation test suite for learning and practice.

## Technologies Used
- Java 21, Maven
- Selenium WebDriver (4.38.0)
- TestNG (7.11.0)
- Allure TestNG (reports)
- Log4j, Apache POI, Rest-assured, jsoup, video-recorder, AspectJ

## Purpose / Objective
- Improve automation skills and gain hands-on experience testing web flows.

## Prerequisites
- Java 21+ and Maven installed
- (Optional) Allure CLI to view reports

## How to Run / Execute
1. Clone the repo:
   git clone https://github.com/esraaatya557/Automation_Exercise_Web.git
2. Change directory:
   cd Automation_Exercise_Web
3. Build and run tests:
   mvn clean test
4. Test results and artifacts are placed under test-output/target (see pom.xml).
